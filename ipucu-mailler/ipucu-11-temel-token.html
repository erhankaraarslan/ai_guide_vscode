Token ve maliyet farkındalığı

İpucu 1

Token nedir?

AI metni 'token' adlı birimlerle işler. Her kelime, noktalama birkaç token. Bağlam penceresi token sayısıyla sınırlı; her mesaj ve eklenen dosya token tüketir. Model seçimine göre (örn. 15K/128K) toplam bağlam değişir.

Nasıl yapılır:

  1. Uzun sohbet = çok token = daha yüksek maliyet
  2. Gereksiz bilgi eklemeyin
  3. Konu değişince yeni sohbet açın

Deneyebileceğiniz örnek:

10+ mesajlık sohbet → /compact veya New Chat

Kaynak: VS Code Docs
İpucu 2

/compact kullanımı

Uzun sohbetlerde geçmiş her mesajda tekrar işlenir. /compact yazınca VS Code konuşmayı özetler; yer açar, maliyet düşer. İsteğe bağlı: '/compact odaklan: veritabanı şeması kararları' ile özeti yönlendirebilirsiniz.

Nasıl yapılır:

  1. Chat giriş kutusuna /compact yazın
  2. Enter'a basın
  3. Konuşma özetlenir, devam edebilirsiniz

Deneyebileceğiniz örnek:

Sohbet 10+ mesaj olduğunda «/compact»

Kaynak: VS Code Docs
İpucu 3

Talimat dosyalarını kısa tutun

copilot-instructions.md her sohbette yüklenir. Uzunsa gereksiz token harcanır. Karar vermeye yardımcı olan bilgiye odaklanın — AI'ın koddan çıkaramayacağı mimari, sözleşmeler, proje kuralları.

Nasıl yapılır:

  1. .github/copilot-instructions.md'de kısa maddeler
  2. Mimari, sözleşmeler, projeye özel kurallar
  3. Genel bilgileri yazmayın

Deneyebileceğiniz örnek:

Mimari kararlar, API prefix, hata formatı
