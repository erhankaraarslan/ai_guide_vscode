Özel ajan derinlemesine

İpucu 1

description ve talimatlar

description: AI'a bu ajanın ne yaptığını anlatır. Altındaki talimatlar her görevde uygulanır. Kısa, net cümleler.

Nasıl yapılır:

  1. description: 1-2 cümle
  2. Talimatlar: maddeler halinde
  3. Örn: 'Planlama ajanı. Kod yazmaz, sadece adım listesi üretir.'

Deneyebileceğiniz örnek:

description: Bu ajan sadece plan üretir. Kod yazmaz.
İpucu 2

tools: tam listesi

search, read, fetch, edit, write, run, runSubagent vb. Her aracın amacı farklı. Görev için minimum araç seçin; güvenlik ve hız için.

Nasıl yapılır:

  1. Dokümantasyonda tools listesini inceleyin
  2. Planlama: search, read, fetch
  3. Uygulama: edit, read, search, run

Deneyebileceğiniz örnek:

İnceleme: read, search. Değişiklik yok.
İpucu 3

Özel ajan test etme

Yeni ajan tanımladıktan sonra basit görevle dene. Yanıt yanlış yöne gidiyorsa description veya talimatları güncelleyin. tools: fazlaysa kısıtlayın.

Nasıl yapılır:

  1. Mod seçiciden yeni ajanı seçin
  2. Basit test: 'Bu projenin yapısını özetle'
  3. Beklenenden farklıysa talimatları netleştirin

Deneyebileceğiniz örnek:

Plan ajanı kod yazıyorsa tools: edit çıkarın
