İpucu 1

Skill nedir?

Talimatlar, scriptler, örnekler içeren klasörler. Özel talimatlardan farkı: sadece ilgili olduğunda yüklenir; her sohbette değil. Token tasarrufu.

Nasıl yapılır:

  1. .github/skills/ klasörü oluşturun
  2. Alt klasör (örn. kod-incelemesi)
  3. SKILL.md ve description ile ne zaman yükleneceğini yazın

Deneyebileceğiniz örnek:

Kod incelemesi skill: 'Kullanıcı code review istediğinde yüklenir'
İpucu 2

.github/skills/ altında SKILL.md

Test, dağıtım, kod incelemesi gibi iş akışları için skill tanımlayın. /skills ile yönetin.

Nasıl yapılır:

  1. Chat'e /skills yazın
  2. Configure Skills ile yeni skill ekleyin
  3. Her skill için alt klasör + SKILL.md

Deneyebileceğiniz örnek:

/skills → Configure Skills → test yazma skill
İpucu 3

Skill'ler taşınabilir

agentskills.io standardında. VS Code, Copilot CLI, GitHub Copilot'ta çalışır. Ekipçe paylaşabilirsiniz.

Nasıl yapılır:

  1. Skill klasörünü projeler arası kopyalayın
  2. .github/skills/ veya ~/.copilot/skills/
  3. Aynı SKILL.md yapısı her yerde geçerli

Deneyebileceğiniz örnek:

Ortak 'güvenlik incelemesi' skill'ini depoda tutun
