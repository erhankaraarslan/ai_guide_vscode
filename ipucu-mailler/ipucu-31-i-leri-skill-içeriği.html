İleri · Skill içeriği

Skill SKILL.md yapısı

İpucu 1

SKILL.md zorunlu alanlar

description: ne zaman yükleneceği. Talimatlar: AI'ın izleyeceği adımlar. Örnekler, scriptler, dosya referansları ekleyebilirsiniz.

Nasıl yapılır:

  1. SKILL.md üstünde description: yazın
  2. Örn: 'Kullanıcı test yazma istediğinde'
  3. Talimatları maddeler halinde ekleyin

Deneyebileceğiniz örnek:

description: Kullanıcı unit test istediğinde bu skill yüklenir
İpucu 2

Skill description ile kapsam

Skill'ler applyTo desteklemez. Ne zaman yükleneceğini description: ile yazın: 'Kullanıcı test yazma istediğinde', 'TypeScript dosyasında çalışırken' gibi. AI sohbet bağlamına göre ilgili skill'i seçer.

Nasıl yapılır:

  1. description: 'Kullanıcı X istediğinde bu skill yüklenir'
  2. Örn: test, refactor, güvenlik incelemesi
  3. Belirsiz description = gereksiz yükleme

Deneyebileceğiniz örnek:

description: Unit test yazma veya test ekleme istendiğinde
İpucu 3

Skill paylaşımı

Skill'leri depo, npm paketi veya ortak klasörde tutun. Ekip üyeleri .github/skills/'e symlink veya kopya ile ekler.

Nasıl yapılır:

  1. Skill klasörünü repo'da tutun
  2. README'de kurulum adımları
  3. Aynı standardı tüm projelerde kullanın

Deneyebileceğiniz örnek:

github.com/ekip/ai-skills repo'sundan skill çek
