Orta · handoffs

Handoff yapılandırma

İpucu 1

handoffs frontmatter

.agent.md üstünde handoffs: bölümü ile geçiş düğmeleri tanımlanır. label: düğme metni, agent: hedef, prompt: gönderilecek mesaj.

Nasıl yapılır:

  1. plan.agent.md'de handoffs: ekleyin
  2. label, agent, prompt alanlarını doldurun
  3. send: true ile otomatik gönderim (isteğe bağlı)

Deneyebileceğiniz örnek:

handoffs: - label: Uygulamaya Geç | agent: impl | prompt: Yukarıdaki plana göre kod yaz
İpucu 2

Planlama → Uygulama handoff

Plan ajanı planı bitirince 'Start Implementation' düğmesi çıkar. Tıklayınca Agent'a plan ve prompt gider.

Nasıl yapılır:

  1. Plan ajanında handoffs tanımlayın
  2. agent: uygulama ajanı adı
  3. prompt: «Plandaki adımları uygula»

Deneyebileceğiniz örnek:

Plan → Start Implementation → Agent
İpucu 3

Uygulama → İnceleme

Uygulama ajanı kodu bitirince 'Start Review' gibi handoff ile inceleme ajana geçebilirsiniz.

Nasıl yapılır:

  1. Uygulama ajanında handoffs
  2. agent: inceleme ajanı
  3. prompt: «Bu değişiklikleri incele»

Deneyebileceğiniz örnek:

Uygulama → Review handoff
