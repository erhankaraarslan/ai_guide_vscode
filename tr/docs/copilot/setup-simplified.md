## VS Code'da Copilot'u Ayarlayın

VS Code'da Copilot kullanmak için GitHub Copilot aboneliğine erişiminiz olması gerekir. Copilot'u doğrudan VS Code içinden ayarlayabilirsiniz.

1. Platformunuz için [Visual Studio Code'u indirin ve yükleyin](https://code.visualstudio.com/Download)

1. VS Code'u başlatın

1. Durum Çubuğundaki Copilot simgesinin üzerine gelin ve **Set up Copilot** seçin.

    ![Durum Çubuğundaki Copilot simgesinin üzerine gelin ve Set up Copilot seçin.](images/setup/setup-copilot-status-bar.png)

1. GitHub hesabınızla giriş yapmak için **Sign in** seçin veya zaten giriş yaptıysanız **Use Copilot** seçin.

    Henüz Copilot aboneliğiniz yoksa, [Copilot Ücretsiz planına](https://docs.github.com/en/copilot/managing-copilot/managing-copilot-as-an-individual-subscriber/managing-copilot-free/about-github-copilot-free) kaydolursunuz.

    > [!IMPORTANT]
    > GitHub Copilot'un ücretsiz sürümünde telemetri şu anda etkindir. Varsayılan olarak, VS Code ve <github.com> deneyiminde kod referansları dahil olmak üzere genel kodla eşleşen kod önerilerine izin verilir. `setting(telemetry.telemetryLevel)` ayarını `off` yaparak telemetri veri toplamadan vazgeçebilir veya [Copilot Ayarları](https://github.com/settings/copilot)'nda hem telemetri hem de kod önerisi ayarlarını ayarlayabilirsiniz.

1. Artık VS Code'da Copilot kullanmaya başlayabilirsiniz. Temelleri [Copilot Hızlı Başlangıç](/docs/copilot/getting-started.md) ile öğrenin.
