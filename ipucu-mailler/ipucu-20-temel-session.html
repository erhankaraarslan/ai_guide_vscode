İpucu 1

İlgisiz geçmişi kaldırın

Konu değiştiyse veya eski sorular geçersizse sohbet geçmişini temizleyin. İlgisiz mesajları silin; Copilot sadece alakalı geçmişi kullansın. Yeni görev için thread ile yeni konuşma açın. New Chat ile yeni oturum en temizi.

Nasıl yapılır:

  1. Mesaja sağ tık → Remove veya New Chat (+) ile yeni sohbet
  2. Bağlam kirlenmişse yeni sohbet açın

Deneyebileceğiniz örnek:

10 mesaj sonra konu değiştiyse → New Chat

Kaynak: GitHub Docs
İpucu 2

Paralel oturumlar

Bağımsız iki göreviniz varsa iki ayrı sohbette ilerleyin. Agent Sessions görünümünden geçiş yapın. Her oturum kendi thread'inde; bağlam karışmaz. /new ile hızlı yeni sohbet.

Nasıl yapılır:

  1. Farklı oturumlar açın
  2. Sol tarafta oturum listesinden geçiş yapın
  3. Her oturum kendi bağlamında çalışır

Deneyebileceğiniz örnek:

Bir sohbet: login. Diğeri: veritabanı. Paralel ilerleyin.

Kaynak: GitHub Docs
İpucu 3

Kontrol noktaları

Ajan yanlış yöne gidiyorsa zincirleme düzeltmek zor olabilir. Checkpoint'e dönerek bilinen iyi duruma geri alın. Incremental complexity: her adımı doğrulayın, karmaşıklık eklemeden önce.

Nasıl yapılır:

  1. Sohbet geçmişinde checkpoint varsa kullanın
  2. O noktaya dönün
  3. Farklı talimatla devam edin

Deneyebileceğiniz örnek:

Ajan 5 dosyayı yanlış değiştirdiyse → checkpoint'a dön

Kaynak: VS Code Context Guide
