Model seçimi

İpucu 1

Görev karmaşıklığına model eşleştirin

Planlama, hata ayıklama, mimari için 'akıl yürütme' ağırlıklı büyük modeller. Basit tamamlama için hızlı küçük modeller yeterli. Auto model seçimi gerektiğinde otomatik geçiş yapar.

Nasıl yapılır:

  1. Chat'te model adına tıklayın
  2. Listeden görevinize uygun modeli seçin
  3. Karmaşık iş → büyük model

Deneyebileceğiniz örnek:

Planlama: Claude Sonnet. Basit düzeltme: daha küçük model.

Kaynak: VS Code Docs
İpucu 2

Uygulama için küçük model

Açık talimatlarla kod yazdırırken küçük modeller hem hızlı hem ucuz olabilir. Context engineering'de implement agent için model: küçük model önerilir. Deneyerek karar verin.

Nasıl yapılır:

  1. .agent.md'de model: ['Model Adı']
  2. Veya Chat'te elle seçin
  3. Aynı istemle farklı modeller deneyin

Deneyebileceğiniz örnek:

model: ['Claude Haiku'] — uygulama ajanı için

Kaynak: VS Code Context Guide
İpucu 3

Prompt/ajanda model sabitleyin

Belirli görevler için .prompt.md veya .agent.md içinde model tanımlayın. Her seferinde seçmek zorunda kalmazsınız.

Nasıl yapılır:

  1. Dosya üstünde model: ['Tercih edilen']
  2. O görev için her zaman bu model kullanılır
  3. Tutarlı sonuç alırsınız

Deneyebileceğiniz örnek:

code-review.prompt.md: model: ['Claude Sonnet']
